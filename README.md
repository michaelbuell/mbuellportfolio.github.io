# Professional Portfolio — Michael Buell

## What Is This?

This is a four-page professional portfolio website built to showcase my education,
work experience, and development projects. It includes a Home page with my resume
content, an About Me page, a Projects page, and a functional Contact page with a
validated, interactive form. The site features a light/dark mode toggle, smooth
scroll behavior, card-based layouts, and hover animations, all styled from a
single shared CSS file with a consistent design system across every page.

## Why Does It Exist?

I built this as a class project for my web development course, but the goal
extends beyond the assignment - I want a real online presence I can point to when
applying for jobs or opportunities that involve the web. It serves as a living
document of my skills and projects that I can continue to build on as I grow as
a developer.

## What Tools Did I Use?

- **VS Code** - my primary code editor, chosen for its widespread use in the
  industry and its strong support for HTML, CSS, and JavaScript
- **GitHub** - used for version control and to manage and track changes across
  the project's files
- **Claude AI** - used to generate, modify, and refine code based on the needs
  of the project, helping me implement features like JavaScript form validation
  that were outside my prior experience
- **Formspree** - a free form backend service that receives contact form
  submissions and forwards them to my email without requiring a custom server
- **Cloudflare Pages** - used to host and deploy the site, chosen for its free
  tier, reliable deployment pipeline, and the ability to serve the site at a
  clean, professional URL
